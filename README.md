# Muhammad Haris Ali Portfolio Website

A clean static portfolio website for Muhammad Haris Ali.

## Files

- `index.html` — website content
- `styles.css` — full responsive styling
- `script.js` — mobile menu and footer year
- `.nojekyll` — keeps GitHub Pages simple for static files

## Before publishing

Replace the contact note or add your real email in the contact section. Do not publish private phone numbers unless you actually want recruiters/clients to have them.

## Deploy on GitHub Pages

1. Create a new public GitHub repository, for example: `haris-portfolio`.
2. Upload all files in this folder to the repository root.
3. Go to repository `Settings` → `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Save.
7. Your website will become available at:
   `https://haris-byte.github.io/haris-portfolio/`

## Better link option
